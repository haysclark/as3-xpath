exec()method
public function exec(xml:XML):*

Execute an XML query on an XML node. The node may be the root node of an XML document or any other XML node. If the node is not the root then relative paths will be evaluated relative to that node; however, absolute paths will still be evaluated from the root of the document to which the node belongs.

execQuery()method 
public static function execQuery(xml:XML, path:String):*

Convenience method. Executes an XPath query on a context. If you plan to execute the same XPath query multiple times, it is more efficient to create an instance of XPathQuery and call exec each time, to prevent the path string being parsed each time. The node may be the root node of an XML document or any other XML node. If the node is not the root then relative paths will be evaluated relative to that node; however, absolute paths will still be evaluated from the root of the document to which the node belongs.

Parameters
xml:XML — The root document or XML node on which to execute the XPath query
 
path:String — The XPath query string

Returns
* — The result of executing the XPath statement. If the statement is a Path expression, this will be an XMLList
