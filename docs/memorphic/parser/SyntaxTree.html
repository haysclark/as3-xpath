Classpublic class SyntaxTree
SubclassesXPathSyntaxTree

SyntaxTree is an abstract base class for Abstract Syntax Tree parsers. In general, create rules which are methods of the form: 
    public function RuleName():Boolean
    {
     startRule("RuleName");
     //....
     nextToken();
     //....
     return match();
    }
Invoke other rules, to determine if a match has been made. Rules must always invoke match() or didntMatch() before returning.
